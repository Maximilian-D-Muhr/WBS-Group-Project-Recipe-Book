## üé¨ Demo Quick Reference

### Setup (Before Presentation)
```bash
npm run dev
# Opens http://localhost:3000
```

### Demo Steps (In Order)
1. **Home Page** (30 sec) - Show navigation
2. **Browse Recipes** (1 min) - Show loading skeleton
3. **Search Feature** (1 min) - Search "pasta" then "chicken"
4. **Recipe Detail** (1 min) - Click a recipe, scroll down
5. **Add to Cookbook** (1 min) - Click button, show success
6. **Cookbook Management** (2 min) - Edit note, save, remove recipe
7. **Mobile Responsive** (30 sec) - Press F12, Ctrl+Shift+M
8. **End Demo** (30 sec) - Stop server, ask questions

**Total Demo Time:** 7 minutes
